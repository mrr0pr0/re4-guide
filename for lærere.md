# Tidlig Deployment – Dev-versjon (Skoleprosjekt)

Dette er en tidlig deployment av utviklingsversjonen relatert til skoleprosjektet. Mye er fortsatt under utvikling, og dette er ikke det endelige produktet.

Filene fungerer slik de gjør i denne versjonen, men dere har ikke tilgang til `.env.local`, noe som betyr at dere ikke kan kjøre:

```bash
npm run dev
```

## Back-end

Backend-tilgang
Dersom dere trenger å se backend:

Dere kan sette opp SQL-databasen selv basert på strukturen som ligger i prosjektet, eller

Dere kan ta kontakt med meg, så kan jeg sende databasetabellene ved behov.
